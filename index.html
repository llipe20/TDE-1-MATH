TRABALHO DISCENTE EFETIVO 1: RELATÓRIO DE RESOLUÇÃO DE EXERCICIOS COM ALGORITMOS E LÓGICA

Atividade apresentada ao curso de Análise e desenvolvimento de sistemas, na disciplina de Matemática Computacional, pelo professor Jefferson de Brito Sousa, como avaliação de aprendizagem de conteúdo.

CAXIAS - MA, 2023

INTRODUÇÃO

A aplicação web em questão foi desenvolvida com o propósito de promover o aprendizado e a aplicação prática de algoritmos e lógica de programação no contexto da resolução de exercícios matemáticos. Este trabalho, realizado como parte de um projeto acadêmico, utiliza estruturas HTML para coletar dados de usuário, aplica estilização através de CSS e implementa algoritmos em JavaScript para realizar cálculos e solucionar problemas matemáticos. Nesta aplicação, exploramos detalhadamente a aplicação e apresentamos os resultados alcançados.

CONCLUSÃO

Resolver exercícios matemáticos por meio de código não apenas aprimorou o nosso entendimento de conceitos, mas também promove o desenvolvimento de habilidades cruciais. A resolução desses problemas por meio de programação foi uma prática bem desafiadora para nós, porém, altamente benéfica, pois aprimorou nossas habilidades analíticas e de programação, enquanto nos prepara para uma variedade de aplicações no mundo real. Esta abordagem promoveu uma compreensão mais profunda dos conceitos matemáticos e aumentou a versatilidade da nossa lógica, tornando-se uma ferramenta valiosa no desenvolvimento da nossa árera de atuação.
